# Fullstack Clean Project: Backend (Java) & Frontend (Next.js)
Este projeto foi desenvolvido para revisitar a stack de Java/spring com foco em Clean Code, com separação de responsabilidades. A arquitetura foi pensada para ser escalável, todo o core é independente.

### Filosofia de Desenvolvimento
Clean Code, SOLID

## Stack:

[Backend] - API Java
Arquitetura: Clean Architecture
Gerenciamento: Maven.
BD: H2
Destaque: Código focado em legibilidade.

[Frontend] - Next.js
Secundario, apenas para consumir a API.

## Estrutura de Pastas

├── backend/  
├── frontend/  
├── .gitignore  
└── README.md

## Como Rodar Localmente
backend -> Execute o Maven, porta 8080.
frontend ->

npm i # instala as dependencias

npm run dev # inicia o servidor

Configuração: Certifique-se de ter o JDK 21 e o Node.js instalados.
